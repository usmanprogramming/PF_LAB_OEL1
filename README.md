# Programming Fundamentals Lab OEL 1 Tasks

This repository contains three simple C programs demonstrating basic programming concepts, including loops, arrays, conditionals, and input/output operations.

---

## Task 1: Online Shopping Cart

**Objective:**  
Simulate a small shopping cart where a customer can buy items based on the money they have.

**Features:**

- Displays a list of items with prices.
- Allows the customer to select items multiple times.
- Checks if the customer has enough money for the selected item.
- Deducts the cost from the available money and confirms the purchase.
- Stops when the customer chooses to checkout or runs out of money.
- Displays the remaining balance at the end.

**Concepts Used:**  
Arrays, loops, conditionals, user input.

## Task 3: Weather Tracking System

**Objective:**  
Track temperatures for a week and calculate the highest, lowest, and average temperature.

**Features:**
